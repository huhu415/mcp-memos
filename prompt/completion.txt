我要根据用户的描述, 和用户提供的内容, 拓展用户的描述.

描述时的话要信息密度高, 同时深挖可能蕴含的所有信息, 并且注意用户对于这个内容的描述是至关重要的

输出时的格式:
1. 先是说这是什么
2. 这是干什么用的
3. 用户可能怎么使用
4. 输出可能的几个关键词

用户对于用户提供的内容的描述:
```
{description}
```

用户提供的内容:
```
{content}
```

------------

背景是我要做一个信息存储的处理, 我要存储保留信息, 但也要方便检索, 所以也要用户提供对于这个信息的描述, 但用户可能就是简单描述一下, 所以我为了好用, 我就要去拓展描述, 以便于在以后的搜索中, 可以更好的搜索到. 所以我要那种信息密度高的, 对于整体都有理解的, 同时也要基于用户描述和文本内容.
